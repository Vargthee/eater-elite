Vibecoded modern, high-performance review platform designed for a specific Gen Z demographic. 


🛠 Tech Stack
Frontend: React 19 + TypeScript (Strict Mode)

Styling: Tailwind CSS + Framer Motion (Micro-interactions)

Database: PostgreSQL (via Supabase)

Logic: PLpgSQL Functions for real-time rating aggregations

Auth: Supabase Auth + Edge Functions for anonymous "Burner" accounts
